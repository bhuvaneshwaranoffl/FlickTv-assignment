# bhuvaneshwaran

FlickTv assignment — Blinkit Money onboarding screen built with Flutter.

## Project

- **Package:** `flicktv_bhuvaneshwaran`
- **App name:** bhuvaneshwaran
- **Architecture:** MVVM (Model – View – ViewModel)
- **Font:** Inter (bundled locally, no third-party font packages)

## Structure

```
lib/features/blinkit_money/
├── domain/models/
├── data/repositories/
└── presentation/
    ├── viewmodels/
    ├── pages/
    └── widgets/
```

## Getting started

```bash
flutter pub get
flutter run
flutter test
```
